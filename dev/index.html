DemoActions.DemoActions — Module

Learning repo for testing various CI and benchmarking tools.

Mainly benchmarking my naive ForwardDiff implementation and Dormand-Prince solver on models I find interesting.

Working through Mathematical Physiology by James Keener and Biomolecular Feedback Systems by Murray.

DemoActions.rk4_inplace_f_prealloc_k! — Method
rk4_inplace_f_prealloc_k!(m, f!, u0, tspan, dt, p)

Explicit fourth-order Runge-Kutta method. Fixed dt. m is mutated inplace and f is an in-place function.

The difference with the other rk4 methods is that the k values are allocated before the loop.

I want to see which of these is fastest.
